# Canny-Edge-Detector
Implementation of the Canny Edge Detector from scratch in Python. Applied on bonn.jpg and compared with OpenCV’s cv2.Canny output using F1-score and Mean Absolute Difference (MAD). Achieved F1 ≥ 0.6 and MAD ≤ 0.07 for accurate edge detection.
